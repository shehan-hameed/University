Full Feature List
📷 Face Recognition Attendance
  • Auto attendance via face recognition camera at door
  • Late detection and absent flagging
  • Cross-referenced with class schedule
🔐 Smart Access Control
  • Solenoid door lock — unlocks on verified face match
  • Time-based auto-locking after class starts
  • Unauthorized entry triggers security alert
🌿 Environment Monitoring & Control
  • DHT22 (temp/humidity), MQ135 (CO₂), BH1750 (light), sound sensor
  • Auto-controls AC, fan, and lights via relay
  • Occupancy-based optimization
🔔 Late & Absence Notification
  • Auto-notifies student via app/SMS if absent
  • Escalates to parent/guardian if threshold exceeded
  • Integrates with university academic warning system
🏃 Evacuation & Emergency Headcount
  • Fire/smoke alarm triggers room occupant snapshot
  • Live headcount dashboard for wardens
  • Missing persons flagging and report generation
🧠 AI & ML Components
  • On-device TinyML face detection (ESP-WHO)
  • Cloud face recognition (AWS Rekognition / DeepFace)
  • Environment optimization model
  • Attendance risk / dropout prediction
  • Anomaly detection for unauthorized access
