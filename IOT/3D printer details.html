01 — Overview

Project Overview

This project involves the complete design and construction of a CoreXY-kinematics FDM (Fused Deposition Modelling) 3D printer with a 300×300mm print bed from the ground up. The team will handle all aspects of the build including mechanical assembly, electronics wiring, firmware configuration (Klipper), and software setup — creating a functional, calibrated machine capable of producing high-quality prints.

Unlike purchasing a commercial printer, this project develops hands-on skills in mechanical engineering, embedded systems, motion control, and open-source software — making it a uniquely educational and original undertaking.

Why CoreXY?

In a CoreXY design, the print bed only moves vertically (Z axis) while the printhead moves on X and Y using a crossed belt system driven by two stationary motors. This dramatically reduces moving mass, enabling higher print speeds, better acceleration, and a larger build volume — ideal for a university-level engineering project that showcases advanced kinematics.

03 — Components

Key Component Groups

🔩

Frame & Motion

2020 aluminium extrusion forms a rigid cubic frame. MGN12 linear rails on all axes provide smooth, precise movement. T8 lead screws drive the Z axis vertically.

⚡

Electronics

BTT SKR Mini E3 V3 controller board with integrated TMC2209 silent stepper drivers. 5× NEMA 17 stepper motors. 24V 15A power supply for high-current bed heating.

🔥

Hotend & Extruder

E3D V6 all-metal hotend supports temperatures up to 260°C. BMG dual-drive extruder provides precise filament control. 5015 blower for part cooling.

🖥️

Firmware & Software

Klipper firmware running on Raspberry Pi 3B+ enables advanced features: input shaping, pressure advance, web interface (Mainsail/Fluidd), and remote monitoring.

📐

Bed System

300×300mm 24V heated bed with PEI spring steel print surface. BLTouch probe enables automatic mesh bed levelling — compensates for any warp or tilt automatically.

🌐

Remote Control

Klipper's web interface (Mainsail) allows full printer control over Wi-Fi from any browser — start prints, monitor temperatures, adjust settings in real time.

08 — Originality

Original Contributions

This project goes beyond following a kit — the team designs, assembles, and configures every system from scratch. Key original contributions include:

⚙️

Custom Mechanical Design

The frame, rail positions, and toolhead mount will be custom designed in CAD to fit the exact 300×300mm build volume — not a copy of any existing design.

🧠

Klipper + Input Shaping

Advanced Klipper firmware features like input shaping (ADXL345 accelerometer) and pressure advance are not standard on most commercial printers — this demonstrates graduate-level tuning.

📡

Remote Monitoring Dashboard

A custom Mainsail/Fluidd web interface with macros, print statistics, and camera monitoring will be implemented and documented as an original software contribution.

📋

Full Build Documentation

A complete assembly manual, wiring diagrams, firmware config files, and calibration guide will be produced — usable by future students as a reference.
